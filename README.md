# Training TypeScript React

Repositorio de ejercicios y templates para el curso de TypeScript.

## Estructura

```
.
├── exercises/          # Enunciados de ejercicios
├── slides/             # Presentaciones del curso
└── templates/          # Templates de proyectos
    ├── caso1/          # Caso 1: Sistema de Tareas
    └── caso2/          # Caso 2: Sistema de Biblioteca
```

## Templates

### Caso 1: Sistema de Tareas
Template con tipos, interfaces y servicios para un sistema de gestión de tareas.

### Caso 2: Sistema de Biblioteca
Template con tipos, clases, repositorios genéricos, servicios de préstamo y validadores.

## Requisitos

- Node.js 18+
- npm 9+
